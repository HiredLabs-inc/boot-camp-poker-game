# Boot Camp Poker Game

A simple, text-based game of Texas Hold'em against AI players.

## Installation

clone the repository

```bash
git clone git@github.com:HiredLabs-inc/boot-camp-poker-game.git && cd boot-camp-poker-game/
```

## Usage

```bash
python3 main.py
```
